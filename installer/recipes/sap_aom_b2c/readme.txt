Asynchronous order management for B2C scenario with SAP ERP

1. Setup the hybris platform
./install.sh -r sap_aom_b2c setup

2. Initialize the hybris platform
./install.sh -r sap_aom_b2c initialize

3. Start the hybris server and datahub
./install.sh -r sap_aom_b2c start

4. Stop the hybris server and datahub
./install.sh -r sap_aom_b2c stop
