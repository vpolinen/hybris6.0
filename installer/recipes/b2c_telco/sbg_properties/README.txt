About the configurations in this folder=========================================Please modify/copy all 3 files in to your Tomcat "lib" directory. This is the default directory used by the core+ web applications.If you are using the recipe then no need to do this manually as the installer does it automatically for you.List of files to copy======================- com.hybris.cis_hybris-cis-mock-subscription-web.properties 	--> Subscription Biling Gateway Service configuration- sbg-subscription.properties 									--> Subscription Biling Gateway Service registration and tenant configuration- com.hybris.cis_hybris-cis-mock-subscription-web-logback.xml	--> Subscription Biling Gateway Service logging configurationNotes======To activate good logging, please adjust the appender to write to a logfile in your location of choice
