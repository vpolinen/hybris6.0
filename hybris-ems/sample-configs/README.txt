About the configurations in this folder=========================================Please modify/copy all 2 files in to your Tomcat "lib" directory. This is the default directory used by the core+ web applications.List of files to copy======================- com.hybris.services.entitlements_entitlements-web.properties --> Entitlement & Metereing Service configuration- com.hybris.services.entitlements_entitlements-web-logback.xml --> Entitlement & Metering Service logging configurationNotes======To activate good logging, please adjust the appender to write to a logfile in your location of choice
